# Open Source Observer (OSO)

Open Source Observer (OSO) is an external analytics platform for understanding
open source health, dependency ecosystems, and project activity.

## How It Relates to `py-libp2p`

- OSO is used for maintainer-level health reporting.
- Dependency graph artifacts can be used as OSO analysis inputs.
- Runtime health stays in `libp2p/rcmgr`; OSO complements it with ecosystem signals.
